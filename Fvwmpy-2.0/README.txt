Fvwmpy is a module interface for controlling and extending the Fvwm
window manager in Python.  It is compatible with Fvwm 2 and Python
2.2.

For more information:

    Fvwmpy
	http://barry.warsaw.us/software/fvwm.html

    Python
	http://www.python.org

    Fvwm
	http://www.fvwm.org

See the Fvwmpy url for usage information.
